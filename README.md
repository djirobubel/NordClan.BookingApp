# NordClan Booking App

Система бронирования переговорных комнат с авторизацией через LDAP и календарём бронирований.

---

## 1. Сборка и запуск

Требуется установленный **Docker** и **Docker Compose**.

Из корня репозитория выполните команду:

docker-compose up --build


Будут подняты сервисы:

API: http://localhost:5000
Клиент (Blazor WASM): http://localhost:8081
PostgreSQL
OpenLDAP + phpLDAPadmin: http://localhost:8080


Чтобы остановить сервисы выполните команду:

docker-compose down


Чтобы полностью очистить данные БД/LDAP:

docker compose down -v
docker compose up --build



## 2. LDAP (кратко)

При запуске через Docker автоматически поднимается OpenLDAP и phpLDAPadmin.

Админ-панель LDAP: http://localhost:8080

Администратор:
- Login DN: cn=admin,dc=nordclan,dc=local
- Password: adminpass

Через phpLDAPadmin необходимо:
- Создать Posix Group users (для того, чтобы создать GID Number для наших пользователей);
- Создать Organisational Unit - Users, в котором создаем пользователей (User Account, например Common Name: Test User1, password: password123 (ssha);
- На фронтенде при входе используем Common Name и password.



## 3. Инструкция по работе

1) Откройте фронт при запуске через Docker http://localhost:8081


2) При первом заходе вы попадаете на /login:

- введите логин/пароль LDAP-пользователя (например, Test User1); 
- при успешном входе вы будете перенаправлены на главную страницу


3) Создание брони:

- выделите интервал в календаре;
- в появившемся окне укажите комнату, название, описание, время начала или окончания (если хотите его изменить);
- нажмите "Сохранить"

Ограничения:
- минимальная длительность — 30 минут;
- максимальная длительность — 8 часов;
- брони одной комнаты не пересекаются


4) Редактирование или удаление:

- нажмите на существующую бронь в календаре;
- в модальном окне можно изменить данные или удалить бронь;
- редактировать/удалять можно только свои брони (по логину пользователя)


5) Время ответа
- внизу страницы отображается время ответа последнего запроса к API.
